◇ 概要 ◇
	Minecraftの起動補助ランチャー
◇ 動作条件 ◇
	OS: Windows Vista,7
	.Net Framework4.0が必要です
◇ ファイル構成 ◇
	McLauncher2.exe (実行ファイル)
	Microsoft.WindowsAPICodePack.dll
	Microsoft.WindowsAPICodePack.Shell.dll
◇ インストール ◇
	Mclauncher2.exeを実行すれば起動します
◇ アンインストール ◇
	フォルダごと消してください。
◇ 使い方 ◇
	いろいろ押して感じて
